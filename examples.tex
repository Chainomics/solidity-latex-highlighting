\section{Introduction}

This \LaTeX includes test source code files to test syntax highlighting for the Solidity programming language.
Solidity is a new language used for programming smart contracts on the Ethereum platform.
The \texttt{solidity-highlighting.tex} file defines the highlighting scheme using \texttt{listings} and \texttt{xcolor} packages.
Solidity syntax highlighting may be useful for researchers and developers who include Solidity source code examples in their papers.
Highlighting helps readers grasp the key concepts with less struggling with unfamiliar syntax.
